# Release Notes — v1.2.0

**Released**: 2026-07-12

**Release Notes for Version 1.2.0**

We are excited to announce the release of version 1.2.0, which brings significant updates to our software. This version focuses on improving how we manage our development processes, making it easier for us to deliver features and fixes, while also enhancing our documentation around these processes. These changes will ultimately help us serve you better.

**Highlights:**

1. **Streamlined Development Workflows:** We have standardized our workflows, shifting to Node 22 LTS. This allows our development team to work more efficiently and reduces potential compatibility issues.

2. **Improved AI Integration:** We have enhanced our capabilities to support artificial intelligence by adding new dependencies that facilitate smoother integration. This means that future updates can leverage advanced AI features more effectively.

3. **Better Release Management:** We introduced a manual dispatch trigger for our release workflows. This gives us more control over when updates go live, ensuring that new features or fixes are only released when they are fully ready.

4. **Changelog Automation:** With the addition of new tools to automate our changelog generation, you can expect clearer and more consistent update notes in the future. This will help you easily track new features and improvements.

5. **Enhanced Documentation:** We've added detailed documentation regarding our development tools and procedures (found in the CLAUDE.md file). This transparency should provide greater insight into how our software is built and managed.

Overall, this release is an important step toward creating a more robust and user-friendly product. As we continue to enhance our software, we appreciate your support and look forward to your feedback!
